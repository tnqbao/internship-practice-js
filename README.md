## Chạy dự án trong môi trường dev

### 1. Cài đặt các package cần thiết
```bash
yarn install
``` 
 hoặc chạy
```bash
npm install
```

### 2. Chạy dự án
```bash
yarn dev
```
 hoặc chạy
```bash
npm run dev
```

### 3. Truy cập vào ứng dụng
Mở trình duyệt và truy cập vào địa chỉ: [http://localhost:3000](http://localhost:5000)
